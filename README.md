# Transformation ve Boosting Karşılaştırması

Bu projede Box-Cox ve Yeo-Johnson dönüşümleri uygulanarak AdaBoost, GradientBoosting, XGBoost ve LightGBM regresyon modellerinin performansları karşılaştırılmıştır.

## Kullanılan Kütüphaneler
- pandas
- scikit-learn
- xgboost
- lightgbm

## Amaç
Dönüşümlerin model başarısına olan etkisini gözlemlemek.
